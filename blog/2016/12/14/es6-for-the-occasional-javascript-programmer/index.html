ES6 for the Occasional JavaScript Programmer

Tags: javascript

Posted on 2016-12-14

A few weeks ago I gave a talk for the Chicago Perl Mongers called "ES6 for the JavaScript Irregular". A lot of developers are only occasional JavaScript developers, and so may not be fully-informed on all the latest developments in the JavaScript world. If this is you, you can read my notes on ECMAScript 6, the latest specification of JavaScript, on my Github.
